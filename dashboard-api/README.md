# Heimdall Dashboard API

NestJS backend API for the Heimdall compliance dashboard.

## Setup

```bash
npm install
```

## Development

```bash
npm run start:dev
```

The API will run on http://localhost:3001

## Production

```bash
npm run build
npm run start:prod
```

## API Endpoints

- `GET /api/dashboard-data` - Get dashboard data
- `GET /api/reports` - Get compliance reports
